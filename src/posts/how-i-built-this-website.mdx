This is the first post on Local Search Ally — and it's a meta one.

I'm documenting the entire process of building this website as a live case study. 
If you're a contractor wondering what goes into a modern website build, this series 
will show you exactly what we do and why we do it.

## The Stack

![Description of the image](/images/technology-stack.jpg)

This site is built with **Next.js 15**, styled with CSS variables, and deployed on **Vercel**. 
The blog you're reading right now is powered by MDX — which means posts are written in 
simple Markdown files, no database required.

## Why Document This?

Two reasons:

1. 1️⃣ Transparency builds trust. I want potential clients to see exactly how I work.
2. 2️⃣ This is a real-world example of the kind of site I build for contractors.
